|| ji ltar chu śiṅ yan lag dag | | ma lus bcas te bśig byas na | |ci yaṅ med ltar skyes bu yaṅ
| | khams bcas bśig na de daṅ ’dra | |

chos rnams thams cad bdag med ces | | de phyir rgyal ba rnams kyis gsuṅs | |khams drug de kun
bdag med par | | khyod la gtan la phab pa yin | |

de ltar bdag daṅ bdag med par | | yaṅ dag ji bźin dmigs su med | |bdag daṅ bdag med lta ba dag
| | de phyir thub pa chen pos bzlog | |

mthoṅ daṅ thos sogs thub pa yis | | bden min brdzun pa min par gsuṅs | |phyogs las mi mthun
phyogs ’gyur na | | de ni gñi ga don du min | |

de ltas dam pa’i don du na | | ’jig rten ’di ni bden brdzun ’das | |de ñid phyir na yaṅ dag
tu | | yod ces med ces źal mi bźes | |

gaṅ źig de ltar rnam kun tu | | yod min de ni kun mkhyen gyis | |mtha’ yod mtha’ med gñi ga
daṅ | | gñis ma yin źes ji skad bstan | |

saṅs rgyas graṅs med gśegs de bźin | | byon gyur pa daṅ da ltar bźugs | |sems can mtha’ ni dus
gsum skyes | | bye ba de dag las ’dod ciṅ | |

’jig rten ’phel ba’i rgyu med la | | zad pa dus gsum du srid na | |thams cad mkhyen pas ji ltar
de’i | | phyin chad luṅ ma bstan par mdzad | |

so so’i skye bo la gsaṅ ba | | gaṅ yin de ni zab mo’i chos | |’jig rten sgyu ma lta bu ñid |
| saṅs rgyas bstan pa bdud rtsi yin | |

ji ltar sgyu ma’i glaṅ po la | | skye daṅ ’jig pa ñid snaṅ yaṅ | |de la don gyi yaṅ dag tu |
| skye daṅ ’jig pa ñid med ltar | |

de bźin sgyu ’dra’i ’jig rten la | | skye daṅ ’jig pa ñid snaṅ yaṅ | |dam pa’i don du skye ba
daṅ | | ’jig pa ñid ni yod ma yin | |

ji ltar sgyu ma’i glaṅ po ni | | gaṅ nas ma ’oṅs gar mi ’gro | |sems rmoṅs tsam du zad pas na
| | yaṅ dag ñid du gnas pa med | |

de bźin sgyu ’dra’i ’jig rten ni | | gaṅ nas ma ’oṅs gar mi ’gro | |sems rmoṅs tsam du zad pas
na | | yaṅ dag ñid du gnas pa med | |

de ltar dus gsum ’das bdag ñid | | tha sñad gdags pa ma gtogs par | |gaṅ źig yod dam med ’gyur
ba’i | | ’jig rten don du ci źig yod | |

saṅs rgyas kyis ni rgyu ’di las | | mtha’ yod pa daṅ mtha’ med daṅ | |gñis daṅ gñis min rnam
pa bźi | | luṅ bstan ma mdzad gźan las min | |

re źig lus ’di mi gtsaṅ ñid | | rags śiṅ mṅon gsum spyod yul ba | |rtag tu snaṅ ba gaṅ gi tshe
| | sems la gnas pa ma yin pa | |

de tshe dam chos gnas med pa | | phra rab mṅon sum ma yin pa | |zab mo de go ji lta bur | |
sems la bde blag ’char bar ’gyur | |

chos ’di zab phyir skye bo yis | | śes par dka’ bar thugs chud de | |des na thub pa saṅs rgyas
nas | | chos bstan pa las log par gyur | |

chos ’di log par śes gyur na | | mi mkhas rnams ni chud kyaṅ za | |’di ltar med par lta ba yi
| | mi gtsaṅ der ni byiṅ bar ’gyur | |

gźan yaṅ de ni log bzuṅ nas | | blun pa mkhas pa’i ṅa rgyal can | |spoṅ bas ma ruṅs bdag ñid
can | | mnar med par ni spyi’u tshugs ’gro | |
