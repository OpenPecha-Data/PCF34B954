ldan pa | | śes rab bzaṅ po bgyid du stsol | |

groṅ daṅ gtsug lag khaṅ dag daṅ | | groṅ khyer kun du ’dun khaṅ daṅ | |chu dkon lam rnams thams
cad du | | chu ra dag ni bgyid du stsol | |

nad pa mgon med sdug bsṅal gyis | | ñam thag dman daṅ ṅan pa dag | |sñiṅ rjes rtag tu yoṅs bzuṅ
źiṅ | | de dag gso phyir gus bsten mdzod | |

dus su ’byuṅ ba’i zas skom daṅ | | bca’ ba ’bru daṅ śiṅ tog daṅ | |de dag sloṅ ba’i skye bo
la | | ma stsal bar du gsol mi rigs | |

lham daṅ gdugs daṅ chu tshags daṅ | | tsher ma dbyuṅ ba’i cha sta daṅ | |khab daṅ skud daṅ bsil
yab dag | | chu ra yi ni gnas su bźag | |

’bras bu gsum daṅ tsha ba gsum | | mar daṅ sbraṅ rtsi mig sman daṅ | |dug sel chu rar bźag bgyi
źiṅ | | grub pa’i sman daṅ sṅags kyaṅ bri | |

lus daṅ rkaṅ pa mgo bsku daṅ | | bal daṅ khri’u daṅ skyo ma daṅ | |bum pa zaṅs daṅ dgra sta
sogs | | chu ra dag tu ’jog tu stsol | |

til daṅ ’bras daṅ ’bru daṅ zas | | bu ram snum daṅ ldan pa yi | |chu ra chuṅ du bsil grib tu
| | ruṅ pa’i chus bkaṅ bgyid du stsol | |

grog sbur tshaṅ gi sgo dag tu | | zas daṅ chu daṅ bum ram daṅ | |’bru yi phuṅ po rtag par ni
| | thugs brtan mi rnams bgyid du stsol | |

bśos gsol sṅon daṅ ’og dag tu | | yi dwags khyi daṅ grog sbur daṅ | |bya la sogs pa ci bde bar
| | rtag tu zas dag stsal bar mdzod | |

gtses pa daṅ ni lo ñes daṅ | | gnon pa daṅ ni yams nad daṅ | |pham gyur yul du ’jig rten pa
| | rjes su bzuṅ ba rgya chen mdzod | |

źiṅ pa ñam thag gyur rnams la | | sa bon zas kyis rjes su zuṅ | |nan gyis dpya khral btaṅ ba
daṅ | | śas rnams kyaṅ ni bskyur ba daṅ | |

chags ñen pa las yoṅs bskyab daṅ | | śo gam btaṅ daṅ śo gam dbri | |de dag sgo na sdod pa yi
| | ñon moṅs las kyaṅ bzlog par mdzod | |

raṅ gi yul daṅ yul gźan gyi | | chom rkun rnams kyaṅ źi bar mdzod | |dbyig gi spogs ni mñam
pa daṅ | | rin thaṅ rigs par bgyid du stsol | |

blon pos ci daṅ ci gsol pa | | ñid kyis kun du mkhyen mdzod la | |’jig rten sman pa gaṅ daṅ
gaṅ | | de daṅ de kun rtag tu mdzod | |

bdag phan ci daṅ ci bya źes | | ji ltar khyed la gus yod pa | |gźan phan ci daṅ ci bya źes |
| de bźin khyod ni gus par mdzod | |

| | ’dod dgur spyad pa ñid du mdzod | |
