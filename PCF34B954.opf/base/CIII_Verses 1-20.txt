|| bsod nams bsam gyis mi khyab las | | ji ltar saṅs rgyas mtshan ’byuṅ ba | | theg pa chen
po’i luṅ chen las | | rgyal po chen po de bźin gson | |

raṅ saṅs rgyas kun las byuṅ daṅ | | slob daṅ mi slob las byuṅ ba | | ’jig rten ma lus kun bsod
nams | | ’jig rten bźin du dpag med gaṅ | |

de bcur bsgres par gyur pa yis | | ba spu’i bu ga gcig ’grub ste | | saṅs rgyas ba spu’i bu
ga ni | | thams cad de daṅ ’dra bas ’byuṅ | |

ba spu’i bu ga thams cad ni | | skyed par byed pa’i bsod nams gaṅ | | de dag brgyar ni bsgyur
ba yis | | dpe byad bzaṅ po gcig tu bźed | |

bsod nams de sñed de sñed kyis | | rgyal po dpe byad bzaṅ po ni | | gcig mthar phyin te de lta
bur | | brgyad bcu’i bar du ’byuṅ bar ’gyur | |

dpe byad bzaṅ po brgyad cu dag | | ’grub pa’i bsod nams tshogs gaṅ yin | | de dag brgyar ni
bsgyur ba yis | | skyes bu chen po’i mtshan gcig go | |

mtshan ni sum cu ’grub pa’i rgyu | | bsod nams rgya chen gaṅ yin pa | | de dag stoṅ du bsgyur
ba yis | | zla ba ña ’dra’i mdzod spu’o | |

mdzod spu yi ni bsod nams dag | | stoṅ phrag brgyar ni bsdoms pa yis | | spyi gtsug bltar ni
mi snaṅ ba | | skyob pa’i gtsug tor skyed par byed | |

de ltar bsod nams tshad med kyaṅ | | ji ltar ’jig rten phyogs kun tu | | bcur bsgres ma lus
brjod pa ltar | | tshad daṅ ldan par bag tsam brjod | |

gaṅ tshe saṅs rgyas gzugs sku yi | | rgyu yaṅ de ltar ’jig rten bźin | | gźal med de tshe chos
sku yi | | rgyu lta ji ltar gźal du yod | |

thams cad kyi ni rgyu chuṅ las | | ’bras bu rgya chen skyed ’gyur na | | saṅs rgyas rgyu ni
dpag med las | | ’bras bu dpag yod bsam thag chad | |

saṅs rgyas rnams kyi gzugs sku ni | | bsod nams tshogs las byuṅ ba ste | | chos kyi sku ni mdor
bsdu na | | rgyal po ye śes tshogs las ’khruṅs | |

de lta bas na tshogs ’di gñis | | saṅs rgyas ñid ni thob pa’i rgyu | | de ltar mdor na bsod
nams daṅ | | ye śes ’di ni rtag brten mdzod | |

rigs pa daṅ ni luṅ bstan pa | | dbugs ’byin byed pa’i rgyu ’di yis | | byaṅ chub bsgrub pa’i
bsod nams ni | | ’di la sgyid lug mi bya’o | |

phyogs rnams kun du nam mkha’ daṅ | | sa daṅ chu daṅ me daṅ rluṅ | | ji ltar mtha’ yas de bźin
du | | sdug bsṅal sems can mtha’ yas ’dod | |

sems can mtha’ yas de dag ni | | byaṅ chub sems dpa’i sñiṅ brtse bas | | sdug bsṅal dag las
draṅ byas te | | saṅs rgyas ñid la ’god par ṅes | |

de ltar brtan par gnas de ni | | mi ñal ba’am ñal yaṅ ruṅ | | yaṅ dag blaṅs pa nas bzuṅ ste
| | bag med gyur kyaṅ sems can rnams | |

mtha’ yas phyir na sems can bźin | | bsod nams mtha’ yas rtag sogs ’gyur | | mtha’ yas des na
saṅs rgyas ñid | | mtha’ yas thob min dka’ źes bya | |

gaṅ źig dpag med dus gnas te | | lus can dpag tu med don du | | byaṅ chub dpag tu med ’dod ciṅ
| | dge ba dpag tu med ’dod pa | |

des ko byaṅ chub dpag med kyaṅ | | dpag tu med pa rnam bźi yi | |tshogs kyis riṅ po mi thogs
par | | ci yi phyir na ’thob mi ’gyur | |
