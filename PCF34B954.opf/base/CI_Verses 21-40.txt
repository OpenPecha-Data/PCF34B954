mi dge ba las sdug bsṅal kun | | de bźin ṅan ’gro thams cad do | |dge las bde ’gro thams cad
daṅ | | skye ba kun tu bde ba dag | |

yid daṅ lus daṅ ṅag gis ni | | mi dge kun las ldog bya źiṅ | |dge la rtag tu ’jug bya ba | |
chos ’di rnam pa gñis su bśad | |

chos ’dis dmyal ba yi dags daṅ | | dud ’gro dag las rnam grol źiṅ | |lha daṅ mi yi naṅ dag tu
| | bde dpal rgyal srid rgyas pa ’thob | |

bsam gtan tshad med gzugs med kyi | | tshaṅs stsogs bde ba myoṅ bar byed | |de ltar mṅon mtho’i
chos ’di daṅ | | de yi ’bras bu mdor bsdus pa’o | |

ṅes par legs pa’i chos dag kyaṅ | | phra źiṅ zab par snaṅ ba daṅ | |byis pa thos daṅ mi ldan
la | | skrag par byed par rgyal bas gsuṅs | |

bdag yod ma yin yod mi ’gyur | | bdag gi yod min mi ’gyur źes | |byis pa dag ni de ltar skrag
| | mkhas pa la ni skrag pa zad | |

skye dgu ’di ni ma lus pa | | ṅar ’dzin pa las byuṅ ba daṅ | |ṅa yir ’dzin ldan skye dgu la
| | phan pa gcig tu gsuṅ bas gsuṅs | |

bdag yod bdag gi yod ces pa | | ’di ni dam pa’i don du log | |gaṅ phyir yaṅ dag ji lta ba |
| yoṅs su śes pas gñis mi ’byuṅ | |

ṅar ’dzin las byuṅ phuṅ po rnams | | ṅar ’dzin de ni don du brdzun | |gaṅ gis sa bon brdzun
pa de’i | | skye ba bden pa ga la źig | |

phuṅ po de ltar mi bden par | | mthoṅ nas ṅar ’dzin spoṅ bar ’gyur | |ṅar ’dzin pa dag spaṅs
nas ni | | phyis ni phuṅ po ’byuṅ mi ’gyur | |

ji ltar me loṅ brten nas su | | raṅ bźin gzugs brñan snaṅ mod kyi | |de ni yaṅ dag ñid du na
| | cuṅ zad kyaṅ ni yod min pa | |

de bźin phuṅ po rnams brten nas | | ṅar ’dzin pa ni dmigs par ’gyur | |raṅ gi bźin gyi gzugs
brñan bźin | | yaṅ dag ñid du de ’ga’ med | |

ji ltar me loṅ ma brten par | | raṅ bźin gzugs brñan mi snaṅ ltar | |phuṅ po rnams la ma brten
par | | ṅar ’dzin pa yaṅ de daṅ ’dra | |

’phags pa kun dga’ bo yis ni | | de lta bu yi don thos nas | |chos la mig ’thob bdag ñid kyis
| | dge sloṅ rnams la bzlas te smras | |

ji srid phuṅ por ’dzin yod pa | | de srid de la ṅar ’dzin yod | |ṅar ’dzin yod na yaṅ las te
| | de las yaṅ ni skye ba yin | |

lam gsum thog mtha’ dbus med pa | | ’khor ba’i dkyil ’khor mgal me yi | |dkyil ’khor lta bur
phan tshun gyi | | rgyu can ’di ni ’khor bar ’gyur | |

de ni raṅ gźan gñis ka daṅ | | dus gsum ñid du’aṅ ma thob phyir | |ṅar ’dzin pa ni zad par ’gyur
| | de nas las daṅ skye ba yaṅ | |

de ltar rgyu ’bras skye ba daṅ | | de dag zad pa ñid mthoṅ nas | |yaṅ dag ñid du ’jig rten la
| | yod med ñid du mi sems so | |

sdug bsṅal thams cad zad pa yis | | chos ’di thos nas rtog med pa | |mi ’jigs gnas la sṅar bdag
| yoṅs su mi śes phyir skrag go | |

mya ṅan ’das la ’di dag kun | | med par gyur na khyod mi ’jigs | |’dir de med par bśad pa la
| | khyod ni ji ste ’jigs par byed | |
