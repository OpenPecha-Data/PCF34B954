rnam śes tshor daṅ ’du śes daṅ | | ’du byed rnams kyaṅ thams cad du | |so sor bdag ñid don med
phyir | | dam pa’i don du don med do | |

ji ltar sdug bsṅal phyir bcos la | | don du bde bar ṅa rgyal byed | |de bźin bde ’joms ’byuṅ
ba la | | sdug bsṅal du yaṅ ṅa rgyal byed | |

de ltar ṅo bo ñid med phyir | | bde daṅ phrad pa’i sred pa daṅ | |sdug bsṅal bral ba’i sred
pa spoṅ | | de phyir de ltar mthoṅ ba grol | |

gaṅ gis sems mthoṅ ’gyur źe na | | tha sñad du ni sems brjod de | |sems byuṅ med par sems mi
’byuṅ | | don med lhan cig mi ’dod do | |

de ltar yaṅ dag ji bźin du | | ’gro ba don med śes nas ni | |rgyu med pa yi me bźin du | | gnas
med len med mya ṅan ’da’ | |

de ltar byaṅ chub sems dpas kyaṅ | | mthoṅ nas byaṅ chub ṅes par ’dod | |de ni sñiṅ rje ’ba’
źig gis | | byaṅ chub bar du srid mtshams sbyor | |

de bźin gśegs pas theg chen las | | byaṅ chub sems dpa’i tshogs bstan na | |de la kun du rmoṅs
pa daṅ | | rab tu sdaṅ bas smod par byed | |

yon tan skyon dag mi śes pa’am | | yon tan skyon du ’du śes pa’am | |yaṅ na yon tan sdaṅ ba
źig | | theg pa che la smod byed graṅ | |

gźan la gnod pa skyon daṅ ni | | gźan la phan pa yon tan du | |śes na theg pa chen po la | |
smod byed yon tan sdaṅ źes brjod | |

raṅ gi don la mi lta źiṅ | | gźan don ro gcig dga’ ba gaṅ | |yon tan ’byuṅ gnas theg chen te
| | des na des sdaṅ bsregs par ’gyur | |

dad pa can ni bzuṅ ñes pas | | cig śos sdaṅ źiṅ khros pa yis | |dad pa can yaṅ bsregs bśad na
| | sdaṅ bas phyir phyogs smos ci dgos | |

dug gis dug ni bsal bya bar | | ji ltar sman dpyad las bśad bźin | |sdug bsṅal gyis kyaṅ mi
phan pa | | bsal bar bya ba ci źig ’gal | |

chos rnams sṅon du yid ’gro źiṅ | | yid ni gtso bo źes grags pas | |sdug bsṅal gyis kyaṅ phan
yid kyis | | phan par byed pa cis mi phan | |

mi bde’aṅ phyis rjes phan ’gyur ba | | bya na bdag daṅ gźan dag la | |bde phan lta źig ci smos
te | | chos ’di gna’ yi lugs yin no | |

gal te bde chuṅ yoṅs btaṅ bas | | bde ba rgya chen mthoṅ ’gyur na | |brtan pas bde chen yaṅs
gzigs la | | bde ba chuṅ du btaṅ bar mdzod | |

gal te de yaṅ mi bzod na | | des na sman pa la sogs pa | |mi źim sman ni sbyin byed pa | | bcom
’gyur ’di ni rtad ma yin | |

gnod par ’gyur ba gaṅ yin de | | la lar mkhas pas phan par mthoṅ | |spyir btaṅ dmigs kyis bsal
ba dag | | bstan bcos dag ni kun la bsṅags | |

theg pa chen po gaṅ źig las | | sñiṅ rje sṅon btaṅ spyod yul daṅ | |ye śes dri ma med bśad pa
| | sems yod su źig de la smod | |

śin tu rgya che rab zab la | | sgyid lug bdag ñid ma sbyaṅs pas | |bdag gźan dgra rnams rmoṅs
pa yis | | ’di na theg pa che la smod | |

sbyin daṅ tshul khrims bzod brtson ’grus | | bsam gtan śes rab sñiṅ rje’i bdag | |theg chen
yin na de yi phyir | | ’di la ñes bśad ci źig yod | |
