yaṅ dag blaṅs pa’i chos rnams ni | | spel bas dpal ldan mdog bzaṅ źiṅ | |źabs kyi loṅ bu mi
mṅon daṅ | | ba spu gyen du phyogs mtshan ’gyur | |

rig daṅ bzo la sogs pa’i gnas | | gus pas len daṅ sbyin pas khyod | |ae na ya yi byin pa daṅ
| | yid rno śes rab chen por ’gyur | |

raṅ gi nor bdog blaṅs gyur na | | myur du btaṅ ba’i brtul źugs kyis | |phyag rgya bde daṅ ’jig
rten gyi | | rnam par ’dren pa ñid du ’gyur | |

gñen bśes phan tshun bye ba dag | | yaṅ dag bsdum par byas pa yis | |dpal ldan mdoms kyis pa
ba ni | | spubs su nub pa’i dam par ’gyur | |

khaṅ bzaṅs gdiṅ ba rgyan dag ni | | bde źiṅ bzaṅ ba rab byin pas | |gser ni btso ma dri med
pa | | śin tu ’jam pa’i mdog ’drar ’gyur | |

bla na med pa’i dbaṅ byin źiṅ | | rigs par bla ma’i rjes ’jug pa | |dpal ldan pa spu ñag re
daṅ | | źal ni mdzod spus brgyan par ’gyur | |

sñan ciṅ dga’ bar smras pa daṅ | | legs par smras daṅ mthun byas pas | |khyod ni dpuṅ mgo zlum
pa daṅ | | ro stod seṅ ge’i ’dra bar ’gyur | |

nad pa rnams la rim gro daṅ | | gsos pas thal goṅ rgyas pa daṅ | |bdag ñid rnal du gnas pa daṅ
| | ro bro ba yi mchog tu ’gyur | |

chos daṅ mthun pa’i bya ba la | | thog draṅs khyod kyi dbu gtsug tor | |legs gnas nya gro dha
lta bur | | chu źeṅ gab pa dag tu ’gyur | |

tshig ni bden pa ’jam po dag | | yun riṅs dus su brjod pa yis | |mi yi dbaṅ po ljags yaṅs śiṅ
| | tshaṅs pa yi ni dbyaṅs ldan ’gyur | |

rtag tu rgyun mi ’chad pa ru | | bden pa’i tshig ni brjod pa yis | |’gram pa seṅ ge’i lta bu
daṅ | | dpal ldan thub pa dga’ bar ’gyur | |

lhag par gus daṅ bkur sti daṅ | | ji ltar rigs par rjes ’jug pas | |tshems ni śin tu dkar ba
daṅ | | mdog ldan mñam pa’i tshems su ’gyur | |

dbyen daṅ phra ma med pa yi | | tshig ni yun riṅs goms pa yis | |dpal ldan tshems ni bźi bcu
tham | | mñam par thags kyaṅ bzaṅ bar ’gyur | |

chags daṅ sdaṅ daṅ rmoṅs med ciṅ | | byams pas sems can bltas pa yis | |spyan ni bkrag can mthon
mthiṅ la | | rdzi ma pa yi lta bur ’gyur | |

de ltar mṅon na rgyur byas pa’i | | sum cu rtsa gñis de dag ni | |skyes bu seṅ ge chen po yi
| | mtshan rnams legs par mkhyen par mdzod | |

dpe byad bzaṅ po brgyad cu ni | | byams pa’i rgyu mthun las byuṅ ba | |gźuṅ ni maṅs su dogs
pa yis | | rgyal po khyod la ma bśad do | |

’khor los sgyur ba thams cad la | | ’di dag yod par ’dod mod kyi | |dag daṅ mdzes daṅ gsal ba
ni | | saṅs rgyas rnams kyi char mi phod | |

’khor los sgyur ba’i mtshan daṅ ni | | dpe byad bzaṅ po gaṅ dag gcig | |thub dbaṅ po la sems
daṅ ba’i | | yan lag gcig las ’byuṅ bśad pa’i | |

bskal pa bye ba brgya phrag ni | | gcig tu bsags pa’i dge pas kyaṅ | |saṅs rgyas ba spu ñag
gcig kyaṅ | | bskyed par nus pa ma yin no | |

ñi ma rnams kyi ’od byed pa | | me khyer rnams daṅ bag ’dra ltar | |saṅs rgyas rnams kyi mtshan
yaṅ ni | | ’khor los sgyur ba rnams daṅ ’dra | |

rin po che’i phreṅ ba las spel ma źes bya ba ste le’u gñis pa’o ||
