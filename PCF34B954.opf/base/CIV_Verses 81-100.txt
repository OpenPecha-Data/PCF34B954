tshul khrims sbyin pas gźan gyi don | | brtson ’grus bzod pas bdag ñid kyi | |bsam gtan śes
rab thar pa’i rgyu | | theg chen don ni bsdus pa yin | |

bdag daṅ gźan phan thar pa’i don | | mdor saṅs rgyas bstan de dag ni | |pha rol phyin drug kho
na yin | | de bas ’di ni saṅs rgyas bka’ | |

byaṅ chub lam chen bsod nams daṅ | | ye śes raṅ bźin saṅs rgyas kyis | |gaṅ las bstan pa’i theg
chen de | | mi śes ldoṅs rnams mi bzod do | |

yon tan mkha’ ltar bsam yas pas | | rgyal ba’i yon tan bsam yas gsuṅs | |gaṅ phyir saṅs rgyas
bdag ñid che | | theg chen las bśad ’dir bzod gyis | |

’phags pa śa ra dwa ti’i bus | | tshul khrims tsam yaṅ mi śes pas | |des na saṅs rgyas bdag
ñid che | | de bsam yas par cis mi bzod | |

theg pa che las skye med bstan | | gźan gyi zad pa stoṅ pa ñid | |zad daṅ mi skye don du ni
| | gcig pas de phyir bzod par gyis | |

stoṅ ñid saṅs rgyas che bdag ñid | | de ltar rigs pas rjes bltas na | |theg chen cig śos las
gsuṅs pa | | mkhas pa rnams la cis mi mñam | |

de bźin gśegs dgoṅs gsuṅs pa rnams | | śes par sla min de yi phyir | |theg gcig theg pa gsum
gsuṅs pas | | btaṅ sñoms kyis ni bdag bsruṅ bya | |

btaṅ sñoms kyis ni sdig mi ’gyur | | sdaṅ bas sdig ’gyur dger mi ’gyur | |de bas theg chen źe
sdaṅ ba | | bdag legs ’dod pas byar mi rigs | |

ñan thos theg pa de las ni | | byaṅ chub sems dpa’i smon lam daṅ | |spyod pa yoṅs bsṅo ma bśad
des | | byaṅ chub sems dpar ga la ’gyur | |

byaṅ chub sems dpa’i byaṅ chub phyir | | byin rlabs saṅs rgyas kyis ma bśad | |don ’di la ni
rgyal ba las | | lhag pa’i tshad ma gźan su yod | |

byin rlabs ’phags pa’i bden don daṅ | | byaṅ chub phyogs mthun ldan pa yi | |lam ni ñan thos
thun moṅ las | | saṅs rgyas ’bras bu gaṅ gis lhag | |

byaṅ chub spyod la gnas pa’i don | | mdo sde las ni bka’ ma stsal | |theg pa che las bka’ stsal
pas | | de phyir mkhas pa rnams kyis bzuṅ | |

brda sprod pa daṅ ji lta bur | | yi ge’i phyi mo’aṅ klog ’jug ltar | |de bźin saṅs rgyas gdul
bya la | | ji tsam bzod pa’i chos ston to | |

kha cig la ni sdig pa las | | rnam par bzlog phyir chos ston te | |kha cig bsod nams ’grub bya’i
phyir | | kha cig la ni gñis brten pa | |

kha cig la ni gñis mi brten | | zab mo khu ’phrigs can ’jigs pa | |stoṅ ñid sñiṅ rje’i sñiṅ
po can | | byaṅ chub bsgrub pa kha cig la | |

de ltar mkhas pas theg chen la | | khoṅ khro ba ni zad bya źiṅ | |rdzogs pa’i byaṅ chub ’grub
bya’i phyir | | lhag par rab tu daṅ bar bya | |

theg pa che la rab daṅ źiṅ | | de las bśad pa spyad pa yis | |bla med byaṅ chub thob ’gyur źiṅ
| | bde ba kun kyaṅ źar la ’grub | |

der ni sbyin daṅ tshul khrims daṅ | | bzod pa’i chos ni khyad par du | |khyim pa la bśad sñiṅ
rje yi | | sñiṅ po can de brtan goms mdzod | |

’on te ’jig rten mi bsrun phyir | | chos kyi rgyal srid bgyid dka’ na | |des na chos daṅ grags
don du | | khyod kyis rab tu byuṅ thob mdzad rigs | |

rin po che’i phreṅ ba las rgyal po’i tshul bstan pa ste le’u bźi pa’o ||
