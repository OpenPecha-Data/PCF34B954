dag | | myur ba ñid du sel bar byed | |

sdig pas ṅan ’gror lus kyis ni | | sdug bsṅal bkres skom la sogs ’byuṅ | |des sdig ma byas bsod
nams kyis | | srid pa gźan na de med do | |

rmoṅs pas yid kyi sdug bsṅal ni | | chags sdaṅ ’jigs daṅ ’dod la sogs | |des rten med pa’i śes
pa yis | | de ni myur ba ñid du spoṅ | |

lus daṅ yid kyi sdug bsṅal gyis | | de ltar śin tu ma smas na | |gal te ’jig rten mthas gtugs
par | | ’jig rten ’dren kyaṅ cis te skyo | |

sdug bsṅal ba la yun thuṅ yaṅ | | bzod dka’ yun riṅ smos ci dgos | |sdug bsṅal med ciṅ bde ba
la | | mtha’ yas dus kyaṅ ji źig gnod | |

de la lus kyi sdug bsṅal med | | yid kyi sdug bsṅal ga la yod | |de ni sñiṅ rjes ’jig rten sdug
| | de ñid kyis ni yun riṅ gnas | |

de bas saṅs rgyas riṅ thogs śes | | blo daṅ ldan pa sgyid mi lug | |ñes zad yon tan don du ni
| | ’di la rtag tu ’bad par bya | |

chags daṅ źe sdaṅ gti mug dag | | ñes par mkhyen nas yoṅs su spoṅs | |ma chags mis daṅ gti mug
med | | yon tan mkhyen nas gus pas brten | |

chags pas yi dags ’gro bar ’gro | | źe sdaṅ gis ni dmyal bar ’phen | |rmoṅs pas phal cher dud
’gror ’gro | | bzlog pas lha daṅ mi ñid do | |

skyon spaṅs yon tan ’dzin pa ni | | mṅon par mtho ba pa yi chos | |śes pas ’dzin pa spaṅs zad
pa | | ṅes par legs pa pa yi chos | |

saṅs rgyas sku gzugs mchod rten daṅ | | gtsug lag khaṅ dag gus tshul du | |śin tu rgya chen
gnas mal sogs | | rgya chen phyug pa bsgrub par mdzod | |

rin chen kun las bgyis pa yis | | saṅs rgyas sku gzugs dbyibs mdzes śiṅ | |legs par bris pa
padma la | | bźugs pa dag kyaṅ bgyid du stsol | |

dam chos dge sloṅ dge ’dun dag | | nan tan kun gyis bskyaṅ bar mdzod | |gser daṅ rin chen dra
ba dag | | ñid kyis mchod rten rnams la thogs | |

gser daṅ dṅul gyi me tog daṅ | | rdo rje byi ru mu tig daṅ | |anda rñil daṅ bai ḍ’u rya | |
mthon ka chen pos mchod rten mchod | |

dam chos smra la mchod pa ni | | rñed daṅ bkur sti daṅ ldan pa | |dga’ bar ’gyur ba rnams bgyi
źiṅ | | chos drug gus par bsten par mdzod | |

bla ma rim gro gus ñan daṅ | | źam riṅ daṅ ni mdoṅs gsol daṅ | |byaṅ chub sems dpa’ rnams la
yaṅ | | rtag tu gus pas mchod par mdzod | |

mu stegs gźan la gus pa yis | | mchod daṅ phyag kyaṅ mi bgyi ste | |mi śes pa rnams de’i rkyen
gyis | | skyon daṅ bcas pa chags ’gyur gda’ | |

thub dbaṅ gsuṅ daṅ des byuṅ ba’i | | gźuṅ rnams bri daṅ glegs bam ni | |snag tsha dag daṅ smyu
gu dag | | sṅon du ’gro ba spyin par mdzod | |

| | ye śes spel ba’i slad du mdzod | |
