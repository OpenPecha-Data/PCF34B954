mi gtsaṅ gaṅ ba’i bum pa ni | | phyi rol bkra yaṅ smad pa yin | |mi gtsaṅ ba yi raṅ bźin lus
| | mi gtsaṅ gaṅ ba cis mi smad | |

gal te khyod ni mi gtsaṅ smod | | dri daṅ phreṅ ba bza’ btuṅ ni | |gtsaṅ ma’aṅ mi gtsaṅ byed
pa gaṅ | | lus ’di ci yi phyir mi smad | |

ji ltar raṅ ṅam gźan dag gi | mi gtsaṅ smad par bya ba gaṅ | |de bźin raṅ daṅ gźan dag gi |
| mi gtsaṅ gzugs ko cis mi smad | |

ji ltar bud med gzugs mi gtsaṅ | | khyod kyi raṅ lus de daṅ ’dra | |de bas phyi daṅ naṅ ñid
la | | ’dod chags bral bar rigs min nam | |

rma dgu dag nas ’dzag pa can | | bdag ñid mṅon sum ’khru byed kyaṅ | |lus mi gtsaṅ par mi rtogs
na | | khyod la bśad pas ci źig phan | |

gaṅ dag mi gtsaṅ lus ’di la | | steg chos sñan dṅags byed pa ni | |ae ma’o gzu lums ae ma’o
blun | | ae ma’o skye bos khrel du ruṅ | |

mi śes mun pas bsgribs pa yi | | sems can ’dir ni phal cher yaṅ | |’dod pa’i don du rtsod ’byuṅ
ba | | mi gtsaṅ don du khyi rnams bźin | |

g-yan pa phrugs na bder ’gyur ba | | de bas g-yen pa med na bde | |de bźin ’jig rten ’dod ldan
bde | | ’dod pa med pa de bas bde | |

gal te de ltar brtags na khyod | | ’dod chags bral ba ma grub kyaṅ | |’on kyaṅ ’dod chags bsrabs
pa yis | | bud med chags par mi ’gyur ro | |

tshe thuṅ ’jigs daṅ sdug bsṅal daṅ | | dmyal ba’i rgyu ni mi bzad pa | |ri dags liṅs te de lta
bas | | rtag tu gsod med brtan por mdzod | |

yan lag thams cad mi gtsaṅ bas | | bskus pa’i sprul gdug ’jigs pa ltar | |gaṅ la brten nas lus
can rnams | | skrag par ’gyur ba de re ṅan | |

char sprin chen po laṅs pa na | | źiṅ pa rnams ni ji lta bar | |gaṅ la brten nas lus can rnams
| | dga’ bar ’gyur ba de re bzaṅ | |

de bas chos min spaṅ bya ste | | g-yel ba med par chos la brten | |bdag ñid daṅ ni ’jig rten
’dis | | bla med byaṅ chub thob ’dod na | |

de yi rtsa ba byaṅ chub sems | | ri dbaṅ rgyal po ltar brtan daṅ | |phyogs mthas gtugs pa’i
sñiṅ rje daṅ | | gñis la mi brten ye śes lags | |

rgyal po chen po skyes chen gyi | | mtshan ni sum cu rtsa gñis kyis | |ji ltar khyod sku brgyan
gyur pa | | de lta bu dag gsan par mdzod | |

mchod rten mchod bya ’phags pa daṅ | | rgan rabs rnams ni legs bkur bas | |dpal ldan phyag źabs
’khor lo yis | | mtshan pa’i ’khor los sgyur bar ’gyur | |

rgyal po chos la rtag par ni | | yaṅ dag blaṅs pa brtan par mdzod | |des ni śin tu źabs gnas
pa’i | | byaṅ chub sems dpar ’gyur ba lags | |

sbyin daṅ sñan pa’i tshig daṅ ni | | phan daṅ don gcig spyod pa yis | |dpal ldan phyag sor dra
ba yis | | ’brel pa’i phyag ris can du ’gyur | |

bza’ ba daṅ ni btuṅ ba’i mchog | | rab tu maṅ po byin pa yis | |dpal ldan phyag daṅ źabs mñen
źiṅ | | sku che bdun yaṅ mtho bar ’gyur | |

’tshe med gsad pa thar byas pas | | sku mdzes draṅ źiṅ che ba daṅ | |tshe riṅ sor mo riṅ ba
daṅ | | rtiṅ pa yaṅs pa dag tu ’gyur | |
