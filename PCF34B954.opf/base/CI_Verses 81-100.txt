skyes bu khams drug ’dus pa’i phyir | | yaṅ dag ma yin ji lta ba | |de bźin khams ni re re la’aṅ
| | ’dus phyir yaṅ dag ñid du min | |

phuṅ po bdag min der de min | | de la de min de med min | |phuṅ bdag me śiṅ ltar ’dres min |
| de phyir bdag ni ji ltar yod | |

’byuṅ gsum sa min ’di la min | | de la ’di min de med min | |re re’aṅ de bźin de yi phyir |
| ’byuṅ ba rnams kyaṅ bdag bźin brdzun | |

sa chu me daṅ rluṅ rnams ni | | re re’aṅ ṅo bo ñid du med | |gaṅ gsum med par re re med | |
gcig med par yaṅ gsum med do | |

gal te gsum med re re med | | gcig med par yaṅ gsum med na | |so sor yod pa ma yin te | | ji
ltar ’dus pa skyed par ’gyur | |

’on te so sor raṅ yod na | | bud śiṅ med par me cis med | |g-yo daṅ thogs daṅ sdud pa daṅ |
| chu daṅ rluṅ daṅ sa de bźin | |

’on te me ’di grags yin na | | ji ltar khyod kyis gsum gźan raṅ | |gsum po rten ciṅ ’byuṅ ba
daṅ | | chos mi mthun par ruṅ ba’aṅ min | |

so sor raṅ yod de dag ni | | ji lta bur na phan tshun yod | |so sor raṅ yod ma yin pa | | de
dag ji ltar phan tshun yod | |

gal te so so raṅ med kyi | | gaṅ na gcig der lhag ma rnams | |ma ’dres pa rnams gcig gnas med
| | ’dres pa so sor raṅ yod min | |

so sor raṅ med ’byuṅ ba rnams | | so so’i raṅ mtshan ga la yod | |so so raṅ med śas cher med
| | mtshan ñid dag ni kun rdzob bśad | |

kha dog dri ro reg pa dag | | de dag la yaṅ tshul ’di yin | |mig daṅ rnam śes gzugs rnams daṅ
| | ma rig las daṅ skye ba daṅ | |

byed pa po las daṅ bya ba graṅs | | ldan daṅ rgyu ’bras dus daṅ ni | |thuṅ daṅ riṅ sogs śes
bya daṅ | | miṅ daṅ miṅ can de bźin no | |

sa daṅ chu daṅ me daṅ rluṅ | | riṅ thuṅ phra daṅ sbom ñid daṅ | |dge sogs ñid ni rnam śes su
| | ’gag par ’gyur źes thub pas gsuṅs | |

rnam śes bstan med mtha’ yas pa | | kun du bdag po de la ni | |sa daṅ chu daṅ me daṅ ni | |
rluṅ gis gnas thod ’gyur ma yin | |

’dir ni riṅ daṅ thuṅ ba daṅ | | phra sbom dge daṅ mi dge daṅ | |’dir ni miṅ daṅ gzugs dag kyaṅ
| | ma lus par ni ’gag par ’gyur | |

gaṅ ma śes phyir rnam śes la | | sṅon chad ’byuṅ ba de kun ni | |de śes phyir na rnam śes su
| | phyis ni de ltar ’gag par ’gyur | |

rnam śes me yi bud śiṅ ni | | ’gro chos ’di kun yin par ’dod | |de dag ji bźin rab ’byed pa’i
| | ’od daṅ ldan pas bsregs nas źi | |

mi śes pa ni sṅon brtags pa | | phyis ni de ñid ṅes pa daṅ | |gaṅ tshe dṅos po mi rñed pa |
| de tshe dṅos med ga la ’gyur | |

gzugs kyi dṅos po miṅ tsam phyir | | nam mkha’ yaṅ ni miṅ tsam mo | |’byuṅ med gzugs lta ga
la yod | | de phyir miṅ tsam ñid kyaṅ yin | |

tshor daṅ ’du śes ’du byed daṅ | | rnam śes ’byuṅ ba lta bu daṅ | |bdag ñid du ni bsam bya ste
| | de phyir khams drug bdag med do | |

rin po che’i phreṅ ba las mṅon par mtho ba daṅ ṅes par legs pa bstan pa ste le’u daṅ po’o ||
