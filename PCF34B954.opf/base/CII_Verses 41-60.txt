phan par smra ba dkon pa ste | | ñan par byed pa śin tu dkon | |de dag bas kyaṅ mi sñan yaṅ
| | phan pa’i rjes su byed pa dkon | |

de lta bas na mi sñan yaṅ | | phan par mkhyen nas myur du spyad | |nad gso’i don du bdag byams
pa | | mi bzad ṅa ba’i sman yaṅ ’thuṅ | |

srog daṅ nad med rgyal srid dag | | mi rtag ñid ni rtag dgoṅs te | |de nas yaṅ dag brtson ldan
par | | chos la gcig tu nan tan mdzod | |

gdon mi za bar ’chi ’gyur źiṅ | | śi nas sdig pas sdug bsṅal ba | |gzigs te ’phral du bde na
yaṅ | | sdig pa bgyi bar mi rigs so | |

la lar ’jigs pa med mthoṅ źiṅ | | dus ’gar ’jigs pa mthoṅ gyur pa | |gal te gcig la yid brtan
na | | gcig la khyod ko cis mi ’jigs | |

chaṅ gis ’jig rten brñas ’gyur źiṅ | | don yal nor yaṅ zad par ’gyur | |rmoṅs pas bya ba ma
yin byed | | de bas chaṅ ni rtag tu spoṅs | |

rgyan po chags daṅ mi dga’ daṅ | | źe sdaṅ g-yo sgyu dgod pa’i gnas | |brdzun daṅ kyal ba tshig
rtsub rgyu | | de bas rtag tu spaṅ bar mdzod | |

bud med chags pa phal cher ni | | bud med gzugs bzaṅ sems las byuṅ | |bud med lus la don du
ni | | gtsaṅ ba cuṅ zad yod ma yin | |

kha ni kha chu rul pa daṅ | | so rkyag mi gtsaṅ snod yin te | |sna ni rnag snabs sna chu’i snod
| | mig ni mig rṅul mchi ma’i snod | |

lto ba’i khoṅ ni bśaṅ gci daṅ | | glo ba mchin sogs snod yin par | |rmoṅs pas bud med ma mthoṅ
ba | | de yi lus la chags par byed | |

lus yul śin tu dri ṅa ba | | chags bral rgyu ni gaṅ yin pa | |de la’aṅ ’jig rten ches chags
na | | gaṅ gi ’dod chags bral bar bkri | |

ji ltar bśaṅ daṅ gci gźi la | | ṅan skyugs phag rab chags pa ltar | |de bźin bśaṅ daṅ gci gźi
la | | ’dod ldan ṅan skyugs pag rnams chags | |

lus kyi groṅ khyer mi gtsaṅ ba | | ’byuṅ ba’i bu ga dod pa gaṅ | |de ni skye bo blun pa yis
| | dga’ ba’i don du ñe bar brtags | |

khyod bdag ñid kyis bśaṅ gci sogs | | so sor mi gtsaṅ mthoṅ nas ni | |de ’dus lus la ji lta
bur | | yid du ’oṅ ba ñid du gyur | |

khrag daṅ khu ba ’dres pa yi | | sa bon mi gtsaṅ sñiṅ pos bskyed | |mi gtsaṅ ṅo bor śes bźin
du | | ’di la ’dod can gaṅ źig chags | |

mi gtsaṅ phuṅ po de’i rlan gyis | | brlan pa’i pags pas g-yogs pa gaṅ | |ñal ba de ni bud med
kyi | | chu so’i khoṅ nas ñal bar zad | |

gzugs bzaṅ ba daṅ gzugs ṅan nam | | rgan nam yaṅ na gźon yaṅ ruṅ | |bud med gzugs kun mi gtsaṅ
na | | khyod chags khyad par gaṅ la bskyed | |

gal te mi gtsaṅ kha dog bzaṅ | | śin tu gsar la dbyibs legs kyaṅ | |de la chags par mi ’os ltar
| | bud med gzugs la’aṅ de bźin no | |

khoṅ rul phyi rol pags pas g-yogs | | ro myags ’di yi raṅ bźin ni | |śin tu mi bzad snaṅ bźin
du | | ji lta bur na mthoṅ ma gyur | |

pags pa ’di yaṅ mi gtsaṅ min | | ral gu bźin du ’dug ce na | |mi gtsaṅ phuṅ po’i pags pa ltar
| | ji lta bur na gtsaṅ mar ’gyur | |
