sna tshogs pa daṅ ’od gsal ba’i | | bgo ba dag daṅ rgyan rnams daṅ | |dri daṅ phreṅ ba loṅs
spyod dag | | sloṅ ba rnams la brtse bas stsol | |

gaṅ la chos don gaṅ med na | | śin tu ñam thag gyur pa de | |de yi mod la bde stsal na | | de
las sbyin mchog ma mchis so | |

gaṅ la dug ni phan ’gyur na | | de la dug kyaṅ sbyin par bgyi | |kha zas mchog kyaṅ mi phan
na | | de la de ni sbyin mi bgyi | |

sbrul gyis zin la ji lta bur | | sor mo bcad la phan bśad pa | |de bźin thub pas gźan phan pa
| | mi bde ba yaṅ bya bar gsuṅs | |

dam pa’i chos daṅ chos smra la | | ñid kyis bkur sti mchog daṅ ni | |gus par byas te chos ñan
źiṅ | | chos kyi sbyin pa dag kyaṅ mdzod | |

’jig rten gtam la dga’ ma mdzad | | ’jig rten ’das la dgyes par mdzod | |bdag la ji ltar de
bźin du | | gźan la yon tan bskyed par mdzod | |

gsan pa’i chos kyis mi ṅoms śiṅ | | don bzuṅ rnam par dbye ba daṅ | |bla ma dag la yon dbul
ba | | rtag tu gus par mdzad du gsol | |

’jig rten rgyaṅ phan sogs mi gdon | | rtsod pa’i don gyi ṅa rgya la spaṅ | |raṅ gi yon tan mi
bsṅags śiṅ | | dgra la’aṅ yon tan brjod par mdzod | |

gnad la dbab par mi bgyi źiṅ | | ṅan sems kyis ni gleṅ mi bgyi | |gźan rnams kyi daṅ bdag ñid
kyi | | ’khrul pa so sor brtag par bya | |

ñes pa gaṅ gis gźan dag la | | mkhas pas rtag tu kha bya ba | |de ni raṅ gis yoṅs spaṅ źiṅ |
| nus pas gźan yaṅ bzlog par mdzod | |

gźan gyis gnod pa bgyis pa la | | sṅon gyi las dgoṅs khro mi bgyi | |sdug bsṅal yaṅ ’byuṅ mi
’gyur bar | | raṅ gi ñes pa med par bgyi | |

lan la re ba ma mchis par | | gźan dag la ni phan par mjod | |sdug bsṅal gcig pus gnag bgyi
źiṅ | | bde ba sloṅ daṅ lhan cig spyad | |

lha yi phun sum tshogs pas kyaṅ | | kheṅs pa ñid du mi bya źiṅ | |yi dags bźin du dbul ba yi
| | rgud pas kyaṅ ni źum mi bgyi | |

bden pa gaṅ gis raṅ [don la] | | rgyal srid ñams par gyur na’aṅ de | |raṅ gi don la rtag brjod
ciṅ | | de las gźan smra mi bgyi’o | |

ji skad gsuṅs pa de bźin du | | mdzad pa’i brtul źugs rtag brten mjod | |des na dpal ldan sa
steṅs su | | tshad ma yi ni dam par ’gyur | |

khyod ni rtag tu thams cad la | | legs par brtags te mdzad pa daṅ | |yaṅ dag ji bźin don gzigs
pas | | gźan gyi driṅ mi ’jog par mdzod | |

chos las rgyal srid bde ba daṅ | | phyogs rnams kun tu grags pa yi | |bla re rgya chen ’byuṅ
’gyur źiṅ | | blon po rnams kyaṅ kun tu ’dun | |

’chi ba’i rkyen ni maṅ ba ste | | gson pa yi ni ñuṅ zad cig | |de dag ñid ni ’chi ba yin | |
de bas rtag tu chos mdzod cig | |

’di ltar rtag tu chos mdzad na | | ’jig rten kun daṅ bdag ñid la | |ñams bder ’gyur ba gaṅ lags
pa | | de ñid re źig ltog mo lags | |

chos kyis bde bar gñid log ciṅ | | bde bar saṅ pa dag tu ’gyur | |naṅ gi bdag ñid skyon med
pas | | rmi lam dag kyaṅ bde ba mthoṅ | |
