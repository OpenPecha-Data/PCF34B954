|| gaṅ phyir bzod dam mi bzod pa | | śes par dka’ pas rgyal po ni | |chos min rigs pa min spyod
kyaṅ | | rjes su ’tsho ba phal cher stod | |

re źig gźan su’aṅ ruṅ ba la ’aṅ | | mi sñan phan pa brjod dga’ na | |sa chen mṅa’ ba’i rgyal
khyod la | | dge sloṅ bdag lta smos ci dgos | |

khyod kyis dgyes pa mdzad slad daṅ | | ’gro ba la yaṅ brtse slad du | |khyod la mi sñan yaṅ
sman pa | | bdag lta gcig pus te por gsol | |

bden ’jam don ldan ’phrod pa ni | | brtse bas dus su slob ma la | |brjod par bya źes bcom ldan
gsuṅ | | de lta bas na de skad brjod | |

brtan pa khro ba med gyur pa | | bden pa’i tshig ni bsgrags pa na | |khrus byed bzaṅ po’i chu
bźin du | | mñan ’os ñams su len par bgyid | |

khyod la bdag gis ’di daṅ ni | | gźan du phan pa brjod bgyid pa | |mkhyen par mdzod de bdag
ñid daṅ | | gźan la sman par ’gyur de mdzod | |

sloṅ la sṅon chad sbyin bgyis pas | | don brñes gyur nas mi sbyin na | |byas mi gzo daṅ chags
pa yis | | phyi nas don rnams ’thob mi ’gyur | |

’jig rten ’di na las byed pa | | gla med lam rgyags khyer mi btub | |sloṅ ba dman pa ma brṅan
par | | phyi ma’i tshen brgya ’gyur khyer | |

rtag tu thugs rje chen po daṅ | | mdzad pa rgya chen las dgyes mdzod | |rgya chen las las ’bras
bu ni | | thams cad rgya chen ’byuṅ bar ’gyur | |

rgyal po dman pa rnams kyis ni | | yid la’aṅ bsams par ma gyur pa’i | |chos gźi dkon mchog gsum
gyi rten | | grags pa dpal daṅ ldan par mdzod | |

chos bźi gaṅ źig kun ’byor pa’i | | rgyal po ba spu mi ldaṅ ba | |śi na’aṅ sñan bar mi ’gyur
bas | | rgyal po de ni ma byas bla | |

rgya rab che phyir rgya chen rnams | | sñems bral spro ba skyed byed ciṅ | |dman rnams spro
ba ’joms byed pa | | bdog pa thams cad gtugs te mdzod | |

khyod ni bdog pa kun bor nas | | dbad med ga śed gśegs ’tshal bas | |chos kyi slad du spyad
pa kun | | khyod kyi mdun du ’doṅ ba ñid | |

mi dbaṅ sṅa ma’i bdog pa kun | | sar pa yis ni dbaṅ gyur na | |sṅa ma’i chos sam bde ba ’am
| | grags pa dag tu ’gyur ram ci | |

nor spyad pa yis ’di la bde | | byin pas gźan du bde bar ’gyur | |ma spyad ma byin chud zos
pas | | sdug bsṅal ’ba’ źig ga la bde | |

’chi khar blon po rjes ṅan can | | gces par mi ’jin rgyal po gsar | |byams par ’dod pa rnams
kyis kyaṅ | | dbaṅ med byas pas sbyin mi spyod | |

de bas rnal gnas myur du ni | | bdog pa kun gyis chos gźi mdzod | |’chi bdag rkyen gyi naṅ gnas
pa | | rluṅ dmar naṅ ’dug mar me bźin | |

sṅon gyi rgyal pos chos kyi gźi | | lha khaṅ la sogs gźan bgyis pa | |gaṅ dag lags pa de dag
kyaṅ | | sṅa lugs bźin du skyoṅ bar mdzod | |

de dag ’tshe med dge spyod ciṅ | | brtul źugs la gnas glo bur byams | |bden bzod ’thab krol
ma mchis pa | | rtag tu brtson rnams spyod du gsol | |

loṅ ba nad pa dman pa daṅ | | mgon med sdug phoṅs yan lag ñams | |de dag kyaṅ ni zas daṅ skom
| | dgag pa med par sñoms thob mdzod | |
