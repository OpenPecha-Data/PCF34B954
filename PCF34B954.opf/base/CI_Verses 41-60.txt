thar la bdag med phuṅ po med | | gal te thar pa de ’drar ’dod | |bdag daṅ phuṅ po gsal ba la
| | ’dir ni khyod ko cis mi dga’ | |

mya ṅan ’das pa dṅos med pa’aṅ | | min na de dṅos ga la yin | |dṅos daṅ dṅos med der ’dzin pa
| | zad pa mya ṅan ’das śes bya’o | |

mdor na med par lta ba ñid | | las kyi ’bras bu med ces pa | |bsod nams ma yin ṅan soṅ ba |
| de ni log par lta źes bśad | |

mdor na yod par lta ba ñid | | las kyi ’bras bu yod ces pa | |bsod nams bde ’gro rgyu mthun
pa | | yaṅ dag lta ba źes bśad do | |

śes pas med daṅ yod źi’i phyir | | sdig daṅ bsod nams las ’das pa | |de yis ṅan ’gro bde ’gro
las | | de ni thar par dam pas bśad | |

rgyu daṅ bcas pas skye mthoṅ bas | | med pa ñid las ’das pa yin | |rgyu daṅ bcas pas ’gag mthoṅ
bas | | yod pa ñid du’aṅ khas mi len | |

sṅar skyes pa daṅ lhan cig skyes | | rgyu min don du rgyu med do | |btags daṅ yaṅ dag ñid du
na | | skye ba’aṅ rab tu ma grags phyir | |

’di yod pas na ’di ’byuṅ dper | | riṅ po yod pas thuṅ du bźin | |’di skyes pas na ’di skye dper
| | mar me skyes pas ’od bźin no | |

riṅ po lta na thuṅ ṅu ñid | | raṅ gi ṅo bo las ma yin | |mar me byuṅ ba med pa’i phyir | | ’on
kyaṅ ’byuṅ ba med pa bźin | |

de ltar rgyu ’bras skye mthoṅ źiṅ | | ’jig rten ’di na ji bźin du | |spros las byuṅ bar khas
blaṅs pas | | med pa ñid du khas mi len | |

’gog pa spros las ma byuṅ ba | | yaṅ dag ji bźin ñid gyur pa | |khas len yod pa ñid mi ’gyur
| | des na gñis mi brten pa grol | |

thag riṅ nas ni mthoṅ ba’i gzugs | | ñe ba rnams kyis gsal bar mthoṅ | |smig rgyu gal te chu
yin na | | ñe ba rnams kyis cis mi mthoṅ | |

ji ltar riṅ ba rnams kyis ni | | ’jig rten ’di ni yaṅ dag mthoṅ | |de ltar de daṅ ñe rnams kyis
| | mi mthoṅ mtshan med smig rgyu bźin | |

ji ltar smig rgyu chu ’dra ba | | chu min don du ma yin pa | |de bźin phuṅ po bdag ’dra ba |
| de dag bdag min don du min | |

smig rgyu la ni ’di chu źes | | bsam de de ni soṅ ba las | |gal te chu de med do źes | | ’dzin
pa de ni blun pa ñid | |

de bźin smig rgyu lta bu yi | | ’jig rten yod pa’am med pa źes | |’dzin pa de ni rmoṅs pa ste
| | rmoṅs pa yod na mi grol lo | |

med pa pa ni ṅan ’gror ’gro | | yod pa pa ni bde ’gror ’gro | |yaṅ dag ji bźin yoṅs śes phyir
| | gñis la mi brten thar par ’gyur | |

yaṅ dag ji bźin yoṅs śes pas | | med daṅ yod par mi ’dod pa | |de phyir med pa par ’gyur na
| | ci phyir yod pa par mi ’gyur | |

gal te yod pa sun phyuṅ bas | | don gyis ’di ni med par bslan | |de bźin med pa sun phyuṅ bas
| | yod par ci yi phyir mi bslan | |

gaṅ dag don gyi med ñid du | | dam mi ’cha’ źiṅ mi spyod pa | |byaṅ chub rten phyir sems med
na | | de dag ji ltar med par bśad | |
