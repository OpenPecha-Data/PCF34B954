de ltar bcu po de dag ni | | byaṅ chub sems dpa’i sa bcur bsgrags | |saṅs rgyas rnams kyi sa
gźan te | | rnam pa kun du gźal yas par | |

rgya che de ni stobs bcu daṅ | | ldan pa yin pa tsam źig brjod | |de yi stobs ni re re yaṅ |
| ’gro ba kun bźin dpag tu med | |

saṅs rgyas rnams kyi tshad med ñid | | phyogs rnams kun gyi nam mkha’ daṅ | |sa chu me rluṅ
ji lta ba | | de tsam źig tu brjod pa bas | |

gal te rgyu ni de tsam du | | tshad med ñid du ma mthoṅ na | |saṅs rgyas rnams kyi tshad med
pa | | de la yid ches mi ’gyur bas | |

de phyir sku gzugs mchod rten gyi | | spyan sṅa ’am yaṅ na gźan yaṅ ruṅ | |tshigs su bcad pa
ñi śu ’di | | ñin gcig bźin yaṅ dus gsum brjod | |

saṅs rgyas chos daṅ dge ’dun daṅ | | byaṅ chub sems dpa’ rnams la yaṅ | |rnam kun btud de skyabs
mchis nas | | mchod ’os rnams la phyag ’tshal lo | |

sdig pa las ni ldog bgyid ciṅ | | bsod nams thams cad yoṅs su bzuṅ | |lus can kun gyi bsod nams
dag | | kun la rjes su yi raṅ ṅo | |

bdag ni spyi btud thal sbyar te | | chos kyi ’khor lo bskor slad daṅ | |’gro gnas bar du bźugs
slad du | | rdzogs pa’i saṅs rgyas rnams la gsol | |

de ltar bgyis pa’i bsod nams daṅ | | bdag gis bgyis daṅ ma bgyis gaṅ | |des ni sems can thams
cad kyaṅ | | bla med byaṅ chub sems ldan śog | |

sems can thams cad dri med dbaṅ | | yoṅs rdzogs mi khom kun ’das śiṅ | |spyod pa raṅ dbaṅ yod
pa daṅ | | ’tsho ba bzaṅ daṅ ldan par śog | |

lus can dag ni thams cad kyaṅ | | lag na rin chen ñid ldan źiṅ | |yo byad thams cad mtha’ yas
pa | | ’khor ba srid du mi zad śog | |

bud med thams cad dus kun tu | | skyes mchog ñid du ’gyur bar śog | |lus can thams cad rig pa
daṅ | | rkaṅ par ldan pa ñid du śog | |

lus can kha dog ldan pa daṅ | | gzugs bzaṅ gzi brjid che ba daṅ | |blta na sdug ciṅ nad med
daṅ | | stobs can tshe daṅ ldan par śog | |

thams cad thabs la mkhas gyur te | | sdug bsṅal kun las thar pa daṅ | |dkon mchog gsum la gźol
ba daṅ | | saṅs rgyas chos nor che ldan śog | |

byams daṅ sñiṅ rje dga’ ba daṅ | | ñon moṅs btaṅ sñoms gnas pa daṅ | |sbyin daṅ tshul khrims
bzod brtson ’grus | | bsam gtan śes rab kyis brgyan ciṅ | |

tshogs rnams thams cad yoṅs rdzogs te | | mtshan daṅ dpe byad gsal ba daṅ | |bsam gyis mi khyab
sa bcu dag | | rgyun mi ’chad par bgrod par śog | |

bdag kyaṅ yon tan de dag daṅ | | gźan kun gyis kyaṅ brgyan ldan te | |ñes pa kun las grol ba
daṅ | | sems can kun mchog byams pa daṅ | |

sems can kun yid re ba yi | | dge ba thams cad rdzogs bgyid ciṅ | |rtag tu lus can thams cad
kyi | | sdug bsṅal sel bar bgyid par śog | |

’jig rten kun na skye bo gaṅ | | su dag ’jigs pas skyo ba rnams | |bdag gi miṅ tsam thos pas
kyaṅ | | śin tu ’jigs pa med par śog | |

bdag ni mthoṅ daṅ dran pa daṅ | | miṅ tsam thos pas skye bo rnams | |rab daṅ ’khrug med rnal
ma daṅ | | rdzogs pa’i byaṅ chub ṅes pa daṅ | |
