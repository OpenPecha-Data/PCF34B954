źig | | yod med ’das pa smra na dris | |

| | ’jig rten don du ci yod dam | |

’das | | don du khyad par ji lta bu | |

| ’gag pa don du ga la źig | |

don du gźan ñid ga la ’gyur | |

phyir | | de rnams gñis ka’aṅ rigs ldan min | |

yaṅ | | rñiṅ pa ’ga’ lta ga la źig | |

ji ltar skad cig mtha’ yod pa | | de bźin thog ma dbus brtag go | |de ltar skad cig gsum bdag
phyir | | ’jig rten skad cig gnas pa min | |

thog ma dbus daṅ tha ma yaṅ | | skad cig bźin du bsam par bya | |thog ma dbus daṅ tha ma ñid
| | raṅ bźan las kyaṅ ma yin no | |

tha dad phyogs phyir gcig pu min | | phyogs med pa ni ’ga’ yaṅ med | |gcig med par ni du ma’aṅ
med | | yod pa med phyir med pa’aṅ med | |

źig pa ’am ni gñen po yis | | yod ñid med par ’gyur graṅ na | |yod pa srid pa ma yin phyir |
| ’jig pa’am gñen por ji ltar ’gyur | |

nam źes | | źus na rgyal ba mi gsuṅ phyir | |

de ltar zab mo’i chos rnams ni | | snod min ’gro la mi gsuṅ ba | |de ñid phyir na mkhas rnams
kyis | | kun mkhyen thams cad mkhyen par śes | |

de ltar ṅes par legs pa’i chos | | zab mo yoṅs su ’dzin med ciṅ | |gnas med ces ni rdzogs saṅs
rgyas | | thams cad gzigs pa rnams kyis gsuṅs | |

pa | | mi mkhas rnams ni phuṅ bar ’gyur | |
